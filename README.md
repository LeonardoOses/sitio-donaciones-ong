# Proyecto Web ONG – Donaciones y Gestión de Eventos

Este proyecto consiste en el desarrollo de una aplicación web para una organización sin fines de lucro, con el objetivo de permitir a los usuarios:

- Consultar información sobre proyectos y campañas en curso.
- Realizar donaciones en línea de forma segura.
- Visualizar notificaciones sobre logros y eventos.
- Participar como voluntarios mediante una futura sección de gestión.

---

## ✅ Buenas prácticas de colaboración implementadas

Durante el desarrollo del proyecto se implementaron las siguientes buenas prácticas en GitHub:

1. **Uso de ramas independientes**
   - Se creó la rama `pagina-donaciones` para trabajar una nueva funcionalidad sin afectar la rama principal.

2. **Pull Requests para revisión de código**
   - Se utilizó un Pull Request para integrar los cambios desde la rama secundaria a `main`, simulando la revisión por parte de colaboradores.

3. **Comentarios constructivos**
   - Se realizaron comentarios en el Pull Request proponiendo mejoras al formulario de donación.

4. **Uso de Issues para discusión**
   - Se creó un issue para proponer la integración de un sistema de gestión de voluntarios, discutiendo su posible implementación.

5. **Mensajes de commit descriptivos**
   - Cada cambio realizado en el código fue acompañado por un mensaje claro que facilita la trazabilidad del historial.

6. **Documentación clara**
   - Se documentaron los avances en este archivo README, y se registraron evidencias del proceso mediante capturas y grabación de video.
